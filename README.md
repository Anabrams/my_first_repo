# Terminal steps for creating a repository:
1. Open the terminal and make sure it is in the correct directory ---> `pwd`
2. Intialize repo, tell gitbash to track.
    - Terminal command: `git init` 
3. Go to github.com and click `new repo` 
4. Give repo a name and hit `create repo`
5. copy link: git remote add origin link ... into vsc terminal 
# Everytime you want to save new data into your github follow these 3 steps:
6. in terminal `git add . `
7. in terminal `git commit -m "name of anything"`
8. in termial `git push origin main` ---> this will push all commits to github
9. Go to github and check your work (refresh the page if you have to)
